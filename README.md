# ADDRESS BOOK

I made a demo project for this address book. The design was inspired by

- [Clay](https://clay.earth/)
- [Less-Annoying-CRM](https://www.lessannoyingcrm.com/)
- [Google-Contat](https://contacts.google.com/)

link to my ptoject: []()

## Features🌟

- **🔍️ Search**: global search such as contacts, notes, to do lists, etc.
- **🏠 Home**: This page contains your contact's recent activities. There is a todo list and a calendar.
- **➕ Add Contact**: will create a new contact form pop up.

  #### Required

        - Name
        - Email
        - Phone

- **📓 Notes**: contains your complete personal notes.

#### tambahkan group 🎮💻🥳

Add groups and contact friends to it. Decorate it by adding your own group icon.
